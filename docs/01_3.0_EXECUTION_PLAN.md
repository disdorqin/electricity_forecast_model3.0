## 阶段 3：day-ahead 接入

来源：

```text
disdorqin/epf-sota-experiment
```

当前默认接入策略：

```text
先接入该仓库的最终可信 champion。
如果该仓库除 cfg05 外还有有效 CatBoost / ensemble 模型，则作为 day-ahead candidate 加入 learner，而不是直接覆盖 champion。
```

注意：

```text
用户称“catboost那个仓库”，但此前资料显示最终可信 champion 为 cfg05 LightGBM = 11.4838%。
因此执行 AI 必须审阅真实报告后确认：
- 哪些模型有效
- 哪些模型作废
- 哪些模型可作为 fusion candidates
```

禁止：

```text
使用 11.27 leakage result
使用 Stage3 old 11.64 natural-day result
使用缺 hour 24 的 690-row 输出
```
